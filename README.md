# Cybersicurity
Este projeto foi desenvolvido no NotebookLM com o objetivo de apresentar, de forma clara e organizada, os principais tipos de ataques cibernéticos utilizados atualmente. O material aborda conceitos, características e exemplos de ataques comuns, como phishing, malware, ransomware, ataques de negação de serviço (DDoS) e engenharia social, contribuindo para a conscientização sobre segurança da informação e boas práticas de proteção no ambiente digital. 🔐💻
